====== Description ========

Rainbow Cloud is a lightweight theme to make beautiful websites with Pagelayer. 
Rainbow Cloud can be used for a blogging site, Business website, personal portfolio, etc. 
It is also specially tailored for Pagelayer so that you can use the drag and drop
features to easily build your website ! It is a simple and fast base theme. We hope you enjoy building your
site with Rainbow Cloud
